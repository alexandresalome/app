Symfony Application
===================

An application skeleton.

Requirements
------------

- PHP8+ with extensions: ctype, iconv
- Node 12+

Technologies used
-----------------

PHP:

- Symfony 5.2
- Doctrine 2.8

Javascript:

- Symfony Encore 1.1
- Symfony Stimulus 2.0
- TypeScript 4.2
- SASS 1.32
- Twitter Bootstrap 5.0

Development
-----------

```
git clone git@github.com:alexandresalome/app.git
cd app
composer install
symfony serve
```
